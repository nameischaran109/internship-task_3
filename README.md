# SQL Developer Internship - Task 3

## Writing Basic SELECT Queries

### Objective

The objective of this task was to practice retrieving data from database tables using SQL SELECT statements and filtering techniques.

### Domain Chosen

Library Management System

### Tools Used

* MySQL Workbench
* GitHub

### Operations Performed

* Retrieved all records using SELECT *
* Retrieved specific columns
* Filtered records using WHERE
* Applied AND and OR conditions
* Used LIKE for pattern matching
* Used BETWEEN for range filtering
* Sorted data using ORDER BY
* Limited results using LIMIT
* Retrieved unique values using DISTINCT
* Used column aliases
